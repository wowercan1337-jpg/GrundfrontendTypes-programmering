README — Todo App in React + TypeScript
Purpose
Create a classic Todo app using React and TypeScript that demonstrates component design and state management with hooks.

Requirements implemented
All parts implemented as React components.
Add new todos using functions and useState.
Written in TypeScript (tsx files, typed props/state).
App includes:
App title.
List displaying all todo items.
Input field for new todo text.
“Add” button to insert new todo into the list.
